任务：基于我提供的产品白底图/透明图，生成淘宝风格“真实买家秀”图片，并自动整理保存到本地桌面。

输入：
- 产品图片路径：{input_image_path}
- 产品名称：{product_name}
- 需要生成组数：{X}

生成规则：

1. 图片结构
- 共生成 X 组
- 每组 5 张图
- 每组必须保持：
  - 同一个人物（年轻亚洲女性）
  - 同一个房间
  - 同一套穿搭
  - 连续拍摄感（像同一次使用过程）

2. 场景风格（必须遵守）
- 淘宝真实买家秀风格
- 禁止商业棚拍感
- 允许：
  - 手机拍摄质感
  - 轻微噪点
  - 轻微糊焦
  - 曝光不稳定
  - 偏黄/偏灰
- 场景：
  - 出租屋 / 宿舍 / 普通卧室 / 小客厅 / 书桌 / 床边
- 动作自然，不要刻意摆拍

3. 产品一致性（强约束）
- 产品外观必须与输入图完全一致：
  - 颜色
  - 结构
  - 按钮
  - 比例
- 禁止任何改动或“再设计”

4. 每组 5 张图内容（固定逻辑）
1）首次试用（开箱或刚上手）
2）换角度继续使用
3）夜间/放松状态使用
4）产品近景或细节补拍
5）随手补图（镜前/床边/随意抓拍）

5. 跨组变化
- 不同组必须更换：
  - 模特
  - 房间
  - 穿搭

6. 一致性校验（必须执行）
- 同组5张：
  - 人脸一致
  - 房间背景一致
  - 光线逻辑一致
- 如果不一致：整组重生成

7. 文件输出

根目录：
~/Desktop/{product_name}/

子目录：
第一组/
第二组/
第三组/ ...

每组内文件命名：
01-首次试用.png
02-继续使用.png
03-细节补拍角度1.png
04-细节补拍角度2.png
05-随手补图.png

8. 续写逻辑
- 如果目录已存在：
  - 自动检测已有“第N组”
  - 从 N+1 继续生成

9. 输出要求
- 自动保存所有图片
- 不输出过程说明
- 最终仅输出一句：
  已生成至：~/Desktop/{product_name}/第N组
